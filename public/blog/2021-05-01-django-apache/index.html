One click deploy Django website with apache server on ubuntu

One click deploy Django website with apache server on ubuntu ...

Get all files here Github repo.

Replace the 'COMPANY' with your project from all files from repo! Copy files to server home Run the following command

sudo chmod u+x deploy_website.sh
./deploy_website.sh

Update settings file

Add ip and domain name to allowed host

ALLOWED_HOSTS = ["*"]

Add "home" to installed app in settings

INSTALLED_APPS = ['home']

Add template DIRS

'DIRS': [os.path.join(BASE_DIR, 'templates')],
